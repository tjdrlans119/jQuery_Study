More_Selector


   Syntax						Description	
1. $(this)						Selects the current HTML element	
2. $("p.intro")					Selects all 'p' elements with class="intro"	
3. $("p:first")					Selects the first 'p' element	
4. $("ul li:first")				Selects the first 'li' element of the first 'ul'	
5. $("ul li:first-child")		Selects the first 'li' element of every 'ul'	
6. $("[href]")					Selects all elements with an href attribute	
7. $("a[target='_blank']")		Selects all 'a' elements with a target attribute value equal to "_blank"	
8. $("a[target!='_blank']")		Selects all 'a' elements with a target attribute value NOT equal to "_blank"	
9. $(":button")					Selects all 'button' elements and 'input' elements of type="button"	
10.$("tr:even")					Selects all even 'tr' elements	
11.$("tr:odd")					Selects all odd 'tr' elements
