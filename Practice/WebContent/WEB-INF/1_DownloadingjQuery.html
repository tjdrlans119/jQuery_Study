jQuery 적용 방법
1. 다운로드 방법: http://jquery.com/download/
	-다운로드한 js파일을 프로젝트에 넣어준다. 사용할 페이지에서 로드시켜준다. 
	 <script src="jquery-3.3.1.min.js" > </script >
	 head에 위 스크립스 jquery를 로드 시켜준다.
2. CDN 사용방법:위와 마찬가지로 페이지에 로그시 킬때 src주소를 cdn 주소를 사용해서 페이지에 로드시킨다.
	-Google CDN : src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"
	-Microsoft CDN : src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.3.1.min.js"
